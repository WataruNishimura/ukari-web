# nuxt-env-vscode

> VSCode 向けの Nuxt 開発環境です。

## Build Setup

``` bash
# 開発環境をインストール
$ yarn install

# テストサーバを立てる。（ アクセスは localhost:3000 ）
$ yarn dev

# サーバを運用する。
$ yarn build
$ yarn start

# 静的サイトを生成
$ yarn generate
```

## 使用しているVSCode拡張機能

- HTML CSS Support
- HTML Snippets
- Prettier
- Pug
- Sass
- SCSS Formatter
- SCSS IntelliSense
- stylelint-plus
- Vetur
- Vue
